# IsoCraft

A Minecraft-inspired isometric voxel game built with TypeScript and Three.js, featuring biome generation powered by [cubiomes](https://github.com/Cubitect/cubiomes).

## Features

- Isometric 3D voxel rendering
- Authentic Minecraft biome generation using cubiomes WASM
- Dynamic music and sound effects
- Pause menu with settings

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Assets

This project requires Minecraft texture and sound assets which are **not included** in this repository due to copyright. You'll need to provide your own assets in the following directories:

- `public/textures/` - Block textures
- `sound-effects/` - Sound effect files
- `soundtrack/` - Music files
- `texturepack/` - Additional texture pack assets
